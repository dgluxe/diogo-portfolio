# Diogo Brasil | Portfólio Pessoal

Um portfólio web responsivo e bilíngue (PT/EN) desenvolvido para apresentar meus projetos, habilidades técnicas e trajetória profissional. O design segue uma estética moderna e minimalista, focada em alta performance e usabilidade.

## Funcionalidades

* Suporte a múltiplos idiomas (Internacionalização via JavaScript) sem recarregar a página
* Estilização dinâmica e otimizada via Tailwind CSS CLI (v4)
* Formulário de contato funcional e assíncrono integrado com Web3Forms
* Interface totalmente responsiva com animações de scroll (Intersection Observer)

## Tecnologias Utilizadas

Para demonstrar versatilidade além do desenvolvimento back-end, este projeto foi construído utilizando:

* HTML5
* Tailwind CSS (v4)
* JavaScript (ES6+)
* NPM / Node.js (Ambiente de build)
